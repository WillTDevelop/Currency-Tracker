# Currency Tracker

Currency Tracker is a simple web app for checking live exchange rates between currencies. You can choose from a wide range of currencies, enter an amount, and instantly see the converted value along with the latest rate.

The app is designed to be clean, lightweight, and easy to use. It fetches real-time data from a currency exchange API and works as a static website, which makes it easy to host on services like GitHub Pages.

## Visit the Website
Go to https://willtdevelop.github.io/Currency-Tracker/

## Open locally

Open `index.html` in your browser.
